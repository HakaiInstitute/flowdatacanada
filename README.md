## Overview

`flowdatacanada` is an R package designed to provide access to spatially
dispersed river flow datasets in Canada through public APIs and data
portals. The package aims to simplify the process of retrieving,
cleaning, and analyzing river flow data for hydrological studies and
environmental monitoring.
